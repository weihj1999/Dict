ECS：
    云硬盘定义：Elastic Volume Service (EVS) disks are a type of virtual block storage devices 
    that use the distributed architecture and can be elastically expanded. You can operate EVS 
    disks online. The method for using EVS disks is the same as that for using common server 
    hard disks. EVS disks have higher data reliability and I/O throughput and are easier to 
    use than common server hard disks. EVS disks can be used for file systems, databases, or 
    system software or applications that require block storage devices.
